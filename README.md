# Caroline Thesis

This repository contains LaTeX tables and related materials for Caroline's thesis on Three-Stage Integrated Framework.

## Contents

- `table_preview.tex` - LaTeX table templates
- `table_preview.html` - HTML preview versions

## Usage

To use the LaTeX tables in your thesis document, simply include the `.tex` files in your main document.

## Preview

Open `table_preview.html` in a browser to preview the table layout.
